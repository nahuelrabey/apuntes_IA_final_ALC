# Introduction to Algorithms (CLRS)

Bienvenido a las notas de lectura de "Introduction to Algorithms". Este documento sirve como punto central para el estudio de uno de los pilares fundamentas de la materia.

## Ficha Bibliográfica

*   **Libro**: [*Introduction to Algorithms*](https://mitpress.mit.edu/9780262033848/introduction-to-algorithms/) (Thomas H. Cormen, Charles E. Leiserson, Ronald L. Rivest, Clifford Stein, 3ª ed., 2009). **MIT Press**. ISBN: 978-0-262-03384-8.

## Objetivo de la Lectura

Este libro es la referencia principal para el análisis de complejidad, estructuras de datos avanzadas y el diseño de algoritmos eficientes en la materia. Se utilizará como base teórica para las demostraciones de correctitud y complejidad.
